# 快速开始

只需添加账号并运行，不必先复制或手写 YAML。

## 1. 解压并打开终端

解压发布包，在 `MihoyoBBSToolsRS.exe` 所在目录右键，选择“在终端中打开”。

Linux 用户将下文的 `.\MihoyoBBSToolsRS.exe` 替换为 `./MihoyoBBSToolsRS`。

## 2. 添加 Cookie，自动创建配置

```powershell
.\MihoyoBBSToolsRS.exe config add-account
```

按提示粘贴完整 Cookie，然后按回车。Cookie 必须包含 UID 和 `stoken`；V2 Stoken 通常还需要 MID。

程序会自动完成以下操作：

- 提取 Stoken；
- 获取米游社昵称；
- 创建 `config/config.yaml`；
- 默认开启原神游戏签到，以及大别野、原神社区签到；
- 默认关闭阅读、点赞和分享。

添加小号时再次执行同一命令。需要备注可使用：

```powershell
.\MihoyoBBSToolsRS.exe config add-account --name "小号"
```

Cookie 只在程序提示后粘贴，不要写进命令参数、截图或发送给他人。

## 3. 运行

```powershell
.\MihoyoBBSToolsRS.exe run
```

以后通常只需执行这一条命令。程序会跳过未绑定的游戏，并只执行配置中已开启、当天尚未完成的任务。

希望双击启动且不等待 BAT 返回时，可生成启动文件：

```powershell
.\MihoyoBBSToolsRS.exe create-launcher
```

生成的 `MihoyoBBSToolsRS-run.bat` 可以移动到桌面或其他位置；它始终指向生成它的 EXE，并以 EXE 所在目录运行。

## 4. 按需设置（可选）

只有需要验证码平台、通知、代理、更多游戏或任务开关时，才打开完整设置菜单：

```powershell
.\MihoyoBBSToolsRS.exe config setup
```

修改后可检查配置：

```powershell
.\MihoyoBBSToolsRS.exe validate-config
```

更多说明见 [使用说明](使用说明.md)。

## 常用命令

| 命令 | 用途 |
|---|---|
| `run` | 执行已开启的签到和米游社任务 |
| `checkin --region china` | 只执行国内游戏签到 |
| `config add-account --name "备注"` | 添加账号 |
| `config remove-account "mys用户:昵称"` | 按生成的账号名删除账号 |
| `run-directory config` | 依次运行配置目录中的多个 YAML |
| `config setup` | 修改全部常用设置 |
| `create-launcher` | 创建可移动的异步启动 BAT（仅 Windows） |
| `validate-config` | 只校验配置 |
| `version` | 查看程序版本 |

升级时直接替换 `MihoyoBBSToolsRS.exe`，不要覆盖 `config/config.yaml`。
