# 快速开始

本页只说明最短使用流程：下载程序、添加 Cookie、运行。通知、代理和全部配置项请看 [完整使用说明](使用说明.md)。

## 1. 下载并完整解压

从 [GitHub Releases](https://github.com/bcmdy/mihoyo-bbs-tools-rs/releases/latest) 下载与系统匹配的文件：

| 系统 | 下载文件 |
|---|---|
| Windows 64 位 | `MihoyoBBSToolsRS-<版本>-windows-x86_64.zip` |
| Linux 64 位电脑/服务器 | `MihoyoBBSToolsRS-<版本>-linux-x86_64.tar.gz` |
| Linux ARM64 | `MihoyoBBSToolsRS-<版本>-linux-arm64.tar.gz` |
| Linux ARMv7 | `MihoyoBBSToolsRS-<版本>-linux-armv7.tar.gz` |

不要只从压缩包预览窗口直接运行程序。完整解压后，配置和日志才能稳定写入程序目录。升级时只替换 `MihoyoBBSToolsRS.exe` 或 `MihoyoBBSToolsRS`，不要覆盖自己的 `config/config.yaml`。

## 2. 在程序目录打开终端

Windows：进入解压目录，在空白处右键选择“在终端中打开”。命令中的 `.` 和反斜杠都需要保留。

Linux：进入解压目录，并授予一次执行权限：

```bash
chmod +x MihoyoBBSToolsRS
```

## 3. 添加 Cookie，自动创建配置

Windows：

```powershell
.\MihoyoBBSToolsRS.exe config add-account
```

Linux：

```bash
./MihoyoBBSToolsRS config add-account
```

看到提示后粘贴完整 Cookie，再按回车。Cookie 需要包含可识别的 UID 和 `stoken`；V2 SToken 通常还需要 MID。

校验成功后，程序会自动：

- 提取 SToken；
- 获取米游社昵称；
- 创建 `config/config.yaml`；
- 使用 `mys用户:<米游社昵称>` 作为账号名；
- 默认开启原神游戏签到，以及大别野、原神社区签到；
- 默认关闭阅读、点赞、取消点赞、分享和其他扩展任务。

添加其他账号时重复执行同一命令。需要备注时使用：

```powershell
.\MihoyoBBSToolsRS.exe config add-account --name "小号"
```

`--name` 是可选备注，最终账号名仍使用米游社昵称。Cookie 只在程序提示后粘贴，不要写在命令后面，也不要截图或发送给他人。

## 4. 运行

Windows：

```powershell
.\MihoyoBBSToolsRS.exe run
```

Linux：

```bash
./MihoyoBBSToolsRS run
```

以后通常只需执行这一条命令。程序只执行配置中已启用、当天尚未完成的任务；没有绑定的游戏会显示 `Skipped`，不算失败。

## 5. 按需设置

需要通知、验证码平台、代理、更多游戏或任务开关时，打开数字设置菜单。Linux 用户将下列 `MihoyoBBSToolsRS.exe` 命令替换为 `./MihoyoBBSToolsRS`：

```powershell
.\MihoyoBBSToolsRS.exe config setup
```

修改后可执行：

```powershell
.\MihoyoBBSToolsRS.exe validate-config
```

Windows 希望双击后独立运行、不等待 BAT 返回时，可生成启动文件：

```powershell
.\MihoyoBBSToolsRS.exe create-launcher
```

生成的 `MihoyoBBSToolsRS-run.bat` 可以移动到桌面或其他目录；它仍会指向生成时的 EXE，并将工作目录设为 EXE 所在位置。移动 EXE 后需要重新生成 BAT。

## 常见问题

### 双击程序一闪而过

这是命令行程序，需要在终端中使用 `run` 等子命令。也可以先运行 `create-launcher`，以后双击生成的 BAT。

### 提示无法识别命令或没有启动

确认程序名与 `run` 之间有空格：

```powershell
.\MihoyoBBSToolsRS.exe run
```

### 无法获取米游社昵称

检查 Cookie 是否完整、是否过期，以及当前网络能否访问米游社。账号尚未添加时，昵称查询不会使用配置中的账号代理。

### YAML 配置无效

先运行 `validate-config` 查看具体行号。手工修改 YAML 时应使用空格缩进，不能使用 Tab；添加通知渠道时不能在 `providers: []` 后继续追加 `- type`。

更多命令、状态码、通知和自动运行说明见 [完整使用说明](使用说明.md)。全部字段见 [YAML 配置参考](configuration.md)。
